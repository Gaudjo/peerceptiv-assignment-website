# Peerceptiv Assignment Website
This repo is for the Peerceptiv Assignment #2 and will demonstrate my competence with HTML structure, CSS styling, and responsive design principles.
<br>
<br>
<br>
<br>
## Project Explanation:
<br>
<br>

<ins>**Why did I structure the HTML the way that I did?:**</ins>

<br>
I structured my HTML setup in a way that follows the natural flow of the page. For each page I first set up the major sections that I would have, such as a header, main and footer sections. I found it easier to start with the bigger chunks of a page to plan out the rough idea of how the page will look and then go back in to fill out the smaller details later.
<br>
<br>
<br>

<ins>**My CSS strategy:**</ins>

<br>
After viewing one of the class tutorial videos from Christina Truong, I learned to love her CSS page layout and have borrowed her techniques. I broke down the CSS into sections using notes for headers. This allows the viewer to know what part of the website is being affected by each section. I keep each section following the page order of the site with global setting and general code snippets coming first. I also tried to keep each class in the order it appears in each section.
<br>
<br>
<br>

<ins>**Accessibility considerations and how I addressed them:**</ins> 

<br>
I made sure to add descriptive alt text for all images on the site and to make sure that all parts of the form on my contact page were properly labeled and connected to their corresponding input. I also tried to be mindful of the use of h elements in my html to allow users to understand the page structure and what elements are in each section.
<br>
<br>
<br>

<ins>**My responsive design strategy:**</ins>

<br>
I went with a mobile first approach. I would keep my screen size small and constantly go to dev mode in my browser to element phone screens. I found this to make building the site easier, as focusing on styling for mobile screens first, meant I had a much easier time styling elements for bigger screens. I used media queries to adjust element sizes and the viewport’s min-width went up. I also used flexbox and Bootstrap to help adjust page content for different screen sizes.
